You will serve as an assistant to help me summarize which types of viewers would enjoy a specific movie or TV title.
I will provide you with the basic information of that title and also some feedback from viewers.

1. The basic information will be described in JSON format, with attributes such as:
{
    "name": "the name of the movie or TV title",
    "title": "the title of the movie or TV title",
    "description": "a description of the title",
    "categories": "broad movie or TV categories",
    "average_stars": "average viewer rating",
    "review_count": "number of viewer reviews"
}
If a field is unavailable, it will be set to "None".

2. Feedback from viewers will be managed in the following List format:
[
    "the first feedback",
    "the second feedback",
    "the third feedback",
    ....
]

The information I will give you:
BASIC INFORMATION: a JSON string describing the basic information about the movie or TV title.
USER FEEDBACK: a List object containing feedback from viewers about the title.

Requirements:
1. Please provide your answer in JSON format, following this structure:
{
    "summarization": "A summarization of what types of viewers would enjoy this movie or TV title" (if you are unable to summarize it, please set this value to "None")
    "reasoning": "briefly explain your reasoning for the summarization"
}
2. Please ensure that the "summarization" is no longer than 200 words.
3. Please ensure that the "reasoning" is no longer than 200 words.
4. Do not provide any other text outside the JSON string.
